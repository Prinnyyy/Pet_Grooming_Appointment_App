# Design Screenshots

This folder stores prototype screenshots used by screenshot-driven Groomly UI tasks.

## Naming

Files use stable ASCII names:

```text
screenshot-YYYY-MM-DD-am-HH-MM-SS.png
screenshot-YYYY-MM-DD-pm-HH-MM-SS.png
```

The names preserve the original screenshot date and clock label while removing spaces and non-ASCII characters from the path.

## Lookup

- Historical task records reference these files from `docs/09_frozen/task_records_2026-06-26/`; current task status is tracked in `docs/06_tasks/TASK_LEDGER.md`.
- Groomly prototype source files remain in `docs/08_design/Groomly/`.
- Design tokens remain in `docs/08_design/design_tokens.json`.
