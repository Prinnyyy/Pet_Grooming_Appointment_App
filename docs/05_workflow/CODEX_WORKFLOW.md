# Codex Workflow

Default to `SINGLE_AGENT_WORKFLOW.md`:

1. one primary task,
2. targeted context only,
3. short plan before non-trivial edits,
4. one focused implementation pass,
5. one mode-appropriate validation attempt,
6. brief diff review,
7. meaningful memory updates only,
8. closeout/checkpoint before `/compact`,
9. concise final report and stop.

Subagents, per-agent reports, and multi-agent review chains are disabled by default.
